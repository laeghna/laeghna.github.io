Beadie's Puzzle Adventure

A puzzle adventure web game.
It is written in HTML5 and (mostly) JavaScript.

The main character is a blob named Beadie.
Beadie has landed in another dimension and must traverse
various worlds, solving puzzles to collect crystals.

Each world has its own puzzle type, and these must be solved
in order to get the crystal for that world.

All crystals must be collected so that Beadie can build a
bridge back home.

Overrun

A zombie survival game created for Android devices.
It is written in Java.

The sole survivor must prevent zombies from breaking through the barrier.
Once the barrier has been breached, one health is lost for each zombie
that passes through. The game ends if a zombie gets to the survivor.
